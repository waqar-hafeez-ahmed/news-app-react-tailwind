# BBC News

A simple BBC news replica that search for news based on search query, or a category selected.

#Technologies

- React.js
- Tailwind
- Vite.js
- Axios

#Features.

- Sidebar.
- Input Field search.
- Category Search.
- Cards displaying news and Images.
